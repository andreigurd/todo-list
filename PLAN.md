# To-Do List Project Plan

## Data Structure
Each task will be a dictionary with:
- description (string)
- priority (High/Medium/Low)
- due_date (string)
- status (Not Started/In Progress/Complete)

## Features
1. Add task
2. View all tasks
3. View by priority
4. View by status
5. Mark complete
6. Delete task
7. Save/load from JSON

## Files
- todo_list.py (main program)
- tasks.json (data storage)
- README.md (documentation)
